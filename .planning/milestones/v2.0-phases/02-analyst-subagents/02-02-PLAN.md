### invest.md の性質

invest.md は **スキルコマンドファイル** である。TypeScript コードではなく、Claude が読んで指示に従うマークダウン文書。Claude は以下のツールを使って指示を実行する:
- **Bash tool**: シェルコマンド実行（mkdir, npx tsx, node -e など）
- **Read tool**: ファイル読み込み（src/agents/*.ts, tmp/*.json など）
- **Agent tool**: サブエージェントのスポーン（5アナリスト並列、モデレーター）

### Agent tool パラメータ

Agent tool には以下のパラメータがある:
- `description` (string): 短いラベル
- `prompt` (string): サブエージェントへのタスク/指示
- `model` (string): "opus", "sonnet", "haiku" など
- `name` (string): アドレス可能な名前
- `run_in_background` (boolean): バックグラウンド実行

**重要:** Agent tool には `schema` パラメータは存在しない（OQ-3, Pitfall 6 参照）。構造化 JSON 出力はプロンプト内の指示で強制する。
