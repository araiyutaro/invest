  <behavior>
    - normalizeUrl: URL をパースし hostname+pathname を返す。パース失敗時は元URL返却。Google NewsのCBMiパスを保持
    - deduplicateByUrl: Map[normalizedUrl, RawNewsArticle] で同一URLをsummary長い方に集約 (D-02)
    - normalizeTitle: NFKC正規化 -> 【】ブラケット除去 -> []ブラケット除去 -> 全角スペース正規化 -> toLowerCase -> trim
    - tokenize: 正規化済みタイトルを空白・句読点で分割 -> 2文字未満除外 -> Set[string] 返却
    - jaccardSimilarity: 交差 / 和集合。両方空なら1.0
    - isJapaneseTitle: タイトルの50%以上が非ASCII(code > 127)なら日本語グループ
    - deduplicateByTitle: 同一言語グループ内のみでO(n^2)比較。Jaccard >= 0.75 (D-01) でsummary長い方を保持 (D-02)。異なる言語グループは比較しない (D-03)
    - filterNewsArticles: Pass1(URL dedup) -> Pass2(Title dedup) を実行し、stats にカウントを記録。Pass3/Pass4 はこの段階ではスキップ (Plan 08-02 で実装)
  </behavior>
  <action>
    filter.ts のスケルトンを実装に置き換える。

    normalizeUrl 関数: new URL(url) で hostname と pathname を結合して返す。catch ブロックで元URL返却。

    deduplicateByUrl 関数: 入力を Map[string, RawNewsArticle] でイテレート。キーは normalizeUrl(article.url)。既存エントリより article.summary.length が長い場合のみ上書き (per D-02)。Map.values() をスプレッドして返却。

    normalizeTitle 関数: title.normalize('NFKC') -> .replace(/【[^】]*】/g, '') -> .replace(/\[[^\]]*\]/g, '') -> .replace(/[　\s]+/g, ' ') -> .toLowerCase() -> .trim()

    tokenize 関数: normalized を /[\s、。,!?]+/ で split し、t.length >= 2 のトークンのみ new Set() に追加。

    jaccardSimilarity 関数: a.size === 0 && b.size === 0 なら 1 を返す。intersection = [...a].filter(x => b.has(x)).length。union = new Set([...a, ...b]).size。union === 0 ? 0 : intersection / union。

    isJapaneseTitle 関数: タイトルの各文字のcharCodeAtを走査し、code > 127 の文字が全体の50%以上なら true (日本語グループ)。

    deduplicateByTitle 関数: 入力配列を走査し、各記事ペアをO(n^2)で比較。isJapaneseTitle() で言語グループ判定し、異なるグループのペアはスキップ (per D-03)。normalizeTitle + tokenize でトークン化し、jaccardSimilarity >= 0.75 (per D-01) なら重複と判定。重複ペアのうち summary.length が短い方を除外マーク (per D-02)。除外されていない記事のみ返却。

    filterNewsArticles 関数: Pass1 deduplicateByUrl -> Pass2 deduplicateByTitle を順次適用。stats オブジェクトに raw, afterUrlDedup, afterTitleDedup を記録。afterRelevance と final は暫定的に afterTitleDedup と同値にする (Plan 08-02 で更新)。

    テストを実行して全 DEDUP テストが GREEN であることを確認してコミットする。
