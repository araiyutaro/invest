## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| resolvedHoldingNews → 静的HTML | resolver が確定した url/title/source が HTML の href 属性・テキストノードに埋め込まれ、docs/ 経由で GitHub Pages に配信される |
| news.json url（元は Finnhub 外部API）→ href 属性 | 記事URLは外部由来。万一 javascript:/data: 等の非http(s)スキームが混入した場合の最終防衛線が必要 |
| LLM出力 HoldingEvaluation.symbol → resolvedHoldingNews キー参照 | portfolio-analyst が出力する symbol は schemas.ts で無正規化（z.string()）。表記揺れがサイレント0件表示を招く境界 |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-20-02 | Tampering | formatHoldingNewsItemHtml（href 埋め込み） | mitigate | safeHref() で http(s) スキームのみ許可、非該当URLはリンク化せずプレーンテキスト表示（reverse tabnabbing の前段防御） |
| T-20-03 | Tampering | ニュース項目の title/source テキスト埋め込み | mitigate | escapeHtml() を title/source を含む全動的文字列に適用（HTMLタグインジェクション防止）。href も escapeHtml 済み |
